# 🎓 Student Marks Analysis System

## 📌 Project Overview

The Student Marks Analysis System is a Python-based project developed to analyze student academic performance using data analysis techniques. The system processes student marks and generates useful insights such as rankings, averages, toppers, and failed students.

## 🚀 Features

- Calculate total marks of each student
- Calculate average marks of each student
- Find topper (highest marks scorer)
- Find lowest marks scorer
- Identify failed students
- Calculate subject-wise average marks
- Generate grades automatically
- Rank students based on performance
- Sort students according to marks
- Find top 3 students
- Find subject-wise maximum marks

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy

## 📊 Analysis Performed

### Student-Level Analysis
- Total Marks
- Average Marks
- Grade Generation
- Student Ranking
- Top 3 Students
- Failed Students

### Subject-Level Analysis
- Subject-wise Average Marks
- Subject-wise Maximum Marks
